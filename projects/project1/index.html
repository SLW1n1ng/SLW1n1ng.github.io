Passenger Booking

The Passenger Booking section allows users to book seats in the airline. There are two classes available: Business Class and Economy Class. Users can click on the respective buttons to select the class they want.

Business Class

When the user clicks on the Business Class button, a form is displayed showing the available seats in the Business Class. The seats are represented as picture boxes, and empty seats are displayed with an empty seat image. Users can click on an empty seat to select it.

If the user hasn't entered their first name and last name in the text boxes provided, an error message is displayed. Once the user enters their name and selects an empty seat, the seat image changes to a full seating image, indicating that the seat has been booked. The user's name is displayed below the seat.

Economy Class

Similar to the Business Class, the Economy Class section allows users to select seats in the Economy Class. The process is the same as the Business Class section.

Database Connection

The project connects to a local database to store and retrieve seat booking information. The connection string is provided in the code, and the database file is specified.

Additional Features

The project includes additional features such as:

  • Canceling a seat: If a user wants to cancel a booked seat, they can click on the seat image. A confirmation message is displayed, and if the user confirms, the seat is cleared and made available for booking again.
  • Airline Menu: The Airline Menu button allows users to go back to the main menu of the airline project.
